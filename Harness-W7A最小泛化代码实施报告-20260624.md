## 2. scripts/harness.js 关键改动

### 2.1 `loadWorkflowConfig(taskId)`（HW7A-P1-002 闭环）

**前**：
- 单例缓存，taskId 切换时仅比较字符串
- 路径硬编码 `weekly-canvas-task.json`
- 无 try/catch
- taskTitle 缺失时 fallback `${taskId} 画布本体UIUX收口`（污染 W6-A 字面值）

**后**：
- cacheKey = `${HARNESS_ROOT}::${taskId}`，taskId 切换必触发完整重 load
- 路径解析：`workflows/weekly-canvas-task-${taskId}.json` 优先，回退 default
- 完整重新解析 JSON + 替换 `_workflowConfig` 模块级引用
- try/catch 处理 JSON 损坏（HW7A-P1-002 / P2-004）
- taskTitle 必填 throw（HW7A-P1-003 闭环 / 方案 A）

### 2.2 `cmdInit` 默认 `fromSubtask`（P2-001 闭环）

**前**：`const fromSubtask = from || 'W6-A-02';`

**后**：取 config 第一个**不在 `firstRunDefaults.importedCompleted` 列表中**的 subtask：
- W6-A：`firstRunDefaults.importedCompleted: ["W6-A-01"]` → 默认 from = W6-A-02 ✓
- W7-A：无 firstRunDefaults → 默认 from = W7-A ✓

### 2.3 `cmdInit` taskTitle 写入

**前**：`taskTitle: getWorkflowConfig().taskTitle || \`${taskId} 画布本体UIUX收口\`,`

**后**：`taskTitle: getWorkflowConfig().taskTitle,`（loadWorkflowConfig 已 throw，config 必填）

### 2.4 W6-A first-run 特殊逻辑抽象（P2-002 闭环）

**前**：
- `const isW6AFirstRun = taskId === 'W6-A' && !from && !stage;`
- `if (isW6AFirstRun || (from === 'W6-A-02' && !stage)) { ... 修改 W6-A-01 risk ... }`
- `if (fromSubtask === 'W6-A-02' && !from && !stage) { console.log('W6-A-01: imported-completed ...') }`

**后**：
```js
const isFirstRun = !from && !stage;
const firstRunDefaults = (getWorkflowConfig().firstRunDefaults) || {};
// ... 在 subtask 初始化循环外：
if (isFirstRun && firstRunDefaults.importedCompleted) {
  for (const sid of firstRunDefaults.importedCompleted) {
    const risk = status.residualRisks.find(r => r.subtask === sid);
    if (risk) {
      risk.risk = `${sid} completed before Harness creation; historical evidence was not re-run by Harness.`;
    }
  }
}
// ... 在 console.log 部分：
if (isFirstRun && firstRunDefaults.importedCompleted) {
  for (const sid of firstRunDefaults.importedCompleted) {
    console.log(`   ${sid}: imported-completed (completed before Harness)`);
  }
}
```

行为完全等价：W6-A JSON 显式注入 `["W6-A-01"]` → W6-A-01 仍标 imported-completed。
